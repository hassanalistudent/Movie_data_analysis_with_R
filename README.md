# Movie Data Analysis with R

## Project Overview  
This repository presents an analytical study of movie datasets using **R programming**. The project highlights skills in data wrangling, visualization, and statistical analysis, with emphasis on deriving meaningful insights from raw data.

## Key Features  
- Data cleaning and preprocessing with `dplyr` and `tidyr`  
- Exploratory data analysis with descriptive statistics  
- Visualizations created using `ggplot2`  
- Comparative analysis across genres and ratings  
- Trend evaluation across multiple years  

## Technology Stack  
- **Language**: R  
- **Libraries**: dplyr, tidyr, ggplot2, readr  
- **Environment**: RStudio  

## Repository Structure  
```
├── data/          # Raw datasets
├── scripts/       # R scripts for analysis
├── plots/         # Generated visualizations
└── README.md      # Documentation
```

## Sample Insights  
- Action films generate higher box office revenue but lower average ratings.  
- Independent films consistently achieve stronger critical acclaim.  
- Ratings trends shifted after 2010, reflecting the influence of streaming platforms.  

## Getting Started  
1. Clone the repository:  
   ```bash
   git clone https://github.com/hassanalistudent/Movie_data_analysis_with_R.git
   ```
2. Open the project in RStudio.  
3. Install dependencies:  
   ```R
   install.packages(c("dplyr","tidyr","ggplot2","readr"))
   ```
